# 🧠 Git Command Guide

🚀 A beginner-friendly guide to mastering **Git**, covering essential commands, concepts, and workflows used in real-world software development.

## 📌 What is Git?

Git is a **distributed version control system** that helps developers:

* Track changes in code
* Collaborate with teams
* Manage project versions efficiently

👉 It allows multiple developers to work on the same project without conflicts ([GeeksforGeeks][1])

## 🛠️ Essential Git Commands

```bash
git init       # Initialize repository
git clone      # Clone repository
git status     # Check status
git add        # Stage changes
git commit     # Save changes
git push       # Upload changes
git pull       # Fetch updates
git log        # View history
```

👉 These commands form the foundation of Git workflows used in real projects ([GeeksforGeeks][2])

## 🌿 Branching & Workflow

* Create branches for features
* Merge code safely
* Work without affecting main code

👉 Branching allows parallel development without breaking the main project ([GeeksforGeeks][1])

## 🌐 Git Workflow

Typical workflow:

```bash
Working Directory → Staging → Commit → Remote Repository
```

* Make changes
* Stage files
* Commit changes
* Push to GitHub
